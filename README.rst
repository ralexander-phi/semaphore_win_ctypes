A ctypes wrapper for Windows Semaphore Objects


* Free software: MIT license
* Documentation: https://semaphore-win-ctypes.readthedocs.io.


Features
--------

Wrappers for the following Windows Semaphore APIs:

* CreateSemaphoreW
* OpenSemaphoreW
* WaitForSingleObject
* ReleaseSemaphore
* CloseHandle
